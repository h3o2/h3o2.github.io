Missing file extensions FIX (Windows 11)

File extensions are essential for identifying different types of files, such as images (.jpg), documents (.docx), and videos (.mp4). However, by default, some versions of the Windows operating system hide these file extensions from view. In this tutorial, we will show you how to reveal or display file extensions in filenames on your Windows computer.

Step 1: Open File Explorer and click on "View"

Step 2: Click on "Show"

Step 3: Click on "File name extensions"

Your settings will be applied, and you should now see file extensions for all known file types.

By following these simple steps, you can easily reveal or show file extensions in filenames on your Windows computer. This feature is helpful when managing different file types and ensuring that you are opening the correct files with the appropriate software.
