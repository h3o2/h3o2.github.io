Create Your First Website with HTML, CSS & JavaScript (Beginner-Friendly)

In this tutorial, I will show you how to create a simple website using HTML, CSS, and JavaScript without needing any specialized tools or access to a computer. We'll use the popular mobile app and website CodePen to write and display your code on your phone for free! By following these steps, you can build your first website from scratch while learning essential web development concepts along the way.

  1. Sign up for CodePen (optional). While you can access CodePen without signing up, creating a free account will allow you to save and share your projects more easily. To sign up, visit https://codepen.io/signup on your phone's web browser.
  2. Create a new pen in CodePen
    1. Open the CodePen website and click "Start Coding" to start a new project or click here: https://codepen.io/pen/. Here are videos demonstrating the process of creating a new Pen when you're not signed in, as well as when you're already signed in.

      Not signed in

      Signed in

    2. Give your project a title, such as "My First Website."
  3. Add basic structure with HTML
    1. In the HTML editor section (<head> tag), add a title element by typing
      <title>My First Website</title>
      This will set the title displayed in the browser's tab or window.
    2. Switch to the main content area (<body> tag) and type
      <h1>Welcome!</h1>
      to create your first heading on the website.
    3. Preview your work by clicking the "Result" button if it's not active already. Now you should see your heading displayed below the HTLM code.
  4. Style your website using CSS
    1. In the editor, click the "CSS" button to open the CSS panel (<style> tag).
    2. Add some basic styling to the element by typing
      h1 {
      blue;
      font-size: 36px;
      }
       This will change the text color and size of your heading. Save your changes, then preview your work again to see the updated appearance.
  5. Add interactivity with JavaScript (optional). While you don't need JavaScript for a basic website, let's add some simple interactivity using CodePen's built-in JavaScript editor.
    1. Click "JS" in the editor panel and type
      document.querySelector( 'h1' ).addEventListener(
      'mouseover', () => {
      document.querySelector( 'h1' ).style.color = 'red';
      }
      );
      This code will change the text color to red when you hover over the heading with your mouse or finger. Save your changes, then preview your work again to see the updated behavior.
  6. Share and save your project. Once you're satisfied with your website, click "Save" at the top right corner of the editor to save your project on CodePen. You can now share a link to your project with friends or family, or continue editing it whenever you want using the same steps outlined above.

Congratulations! You have created your first website using HTML, CSS, and JavaScript on your phone without needing access to a computer. Keep practicing and exploring new concepts to build more advanced websites in the future.
